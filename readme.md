## üü£ Phase 1: Types & Coercion

> [!IMPORTANT]
> **LEVEL: FUNDAMENTAL | THE ARCHITECTURE OF TYPES**
>
> - **Primitive Mechanics:** Undefined, Strings, Numbers, Booleans, Objects, and Symbols.
> - **Special Values:** Deep dive into `NaN`, negative zero, and strict type checking.
> - **Coercion:** Abstract operations, implicit vs. explicit coercion, and mastering loose (`==`) vs. strict (`===`) equals.
>
> _Milestone: Type System Mastery_

<br/>

## üü£ Phase 2: Lexical Scope & Closures

> [!TIP]
> **LEVEL: ADVANCED | COMPILER PATTERNS**
>
> - **Lexical Scope:** How the JS engine compiles and executes code, LHS/RHS lookups.
> - **Hoisting Engine:** Variable declarations, function expressions, and the Temporal Dead Zone (TDZ).
> - **Closures & Modules:** Preserving state across lexical boundaries and building self-documenting module patterns.
>
> _Milestone: The Module Pattern Engine_

<br/>

## üü£ Phase 3: Objects & Prototypes

> [!CAUTION]
> **LEVEL: EXPERT | THE `this` ENGINE**
>
> - **The `this` Keyword:** Implicit, explicit, default, and hard binding rules.
> - **Prototypes:** The prototype chain, shadowing, and `[[Prototype]]` linkage.
> - **OLOO Pattern:** Objects Linked to Other Objects vs. classical Class architectures.
>
> _Milestone: Architecting Scalable Objects_
